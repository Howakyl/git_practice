CSS Property Reference

Property Description
font-family: Specifies the font of an element. MDN
font-size: Specifies the size of an element. MDN
font-weight: Determines how thick or thin an element is. MDN
text-align: Aligns text to the left, center, or right end of the page. MDN
color: Specifies the foreground color of an element. MDN
background-color: Specifies the background of an element. MDN
opacity: Specifies the transparency of an element. MDN
background-image: Sets one or more background images for an element. MDN
